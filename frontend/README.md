# Frontend

## Development Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Run tests:
```bash
npm run test:unit
npm run test:e2e
```

## Tech Stack

- Vue.js 3 with Composition API
- TypeScript
- Pinia for state management
- Vue Router
- Tailwind CSS
- Vite for build tooling
- Vitest for unit testing
- Playwright for E2E testing
